doodle-css实例使用步骤：

  1. 登录doodle-css官网：https://css-doodle.com/

  2. 引入css-doodle文件

        <script src="https://cdnjs.cloudflare.com/ajax/libs/css-doodle/0.4.9/css-doodle.min.js"></script>

  3. 在body标签中插入标签：

        <css-doodle>
      /* put your code here */
    </css-doodle>

  4. 在css-doodle标签中写入doodle的根样式代码。

    :doodle{
    @grid: 1x10 / 61.8vmin;//意思是一行十列
    background:pink;
}

  5. 将页面在浏览器中打开，就会看到一个一行10列的表格。

  6. 为了看到的效果更明显，我们将背景去掉，换成border，使用CSS_WorkFlow来写。

    border:calc(@index()*1vmin) hsla(calc(10*@index()), 60%, 50%, calc(5/ @index()*.8)) dotted ;

  7. 这样写出的格子默认是从左到右排列的，为了好看，改变一下盒子的排列方向，在代码中加入：

    @place-cell: center;
border-radius:50%;

  8. 这时会看到一个类似靶心的东西，我们还可以将边框设成虚线并设置一个动画来让圆做旋转运动。

    --rd:@rand(-360deg);
--d:@rand(2s,4s);
--rt:calc(var(--rd) + @pick(1turn,-1turn));
animation: move var(--d) linear infinite;
@keyframes move {
    0%{
            transform: rotateZ(var(--rd));
            opacity: 1;

    }
    100%{
            transform: rotateZ(var(--rt));
            opacity: 1;
    }

  9. 这时一个运动的圆就诞生了，最终代码如下：

    <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <script src="https://cdnjs.cloudflare.com/ajax/libs/css-doodle/0.4.9/css-doodle.min.js"></script>
    <title>Document</title>
    <style>
        body,
        html {
            height: 100%;
            width: 100%;
            background: gray;
            display: flex;
            margin: 0;
            align-items: center;
            justify-content: center;
            overflow: hidden;
        }
    </style>
</head>

<body>
    <css-doodle>
        :doodle{
            @grid: 1x10 / 61.8vmin;
        }
        @place-cell: center;
        @size:calc(@index()*10%);
        border:calc(@index()*1vmin) hsla(calc(10*@index()), 60%, 50%, calc(5/ @index()*.8)) dotted ;
        border-radius:50%;
        --rd:@rand(-360deg);
        --d:@rand(2s,4s);
        --rt:calc(var(--rd) + @pick(1turn,-1turn));
        animation: move var(--d) linear infinite;
        @keyframes move {
            0%{
                    transform: rotateZ(var(--rd));
                    opacity: 1;

            }
            100%{
                    transform: rotateZ(var(--rt));
                    opacity: 1;
            }

    }
    </css-doodle>
</body>

</html>

houdini：代码实战

  1. 新建houdini.html文件

  2. 判断浏览器是否支持CSS.paintWorklet，如果不支持，输出一句话

    if(!CSS in window || !CSS.paintWorklet){
    console.log("你用的是啥浏览器呀，都不支持CSS.paintWorklet");
}else{
    CSS.paintWorklet.addModule("sky.js");

  3. 插入style标签，编写css样式

        margin:0;
        color:white;
        background:black;
    }
    body::before{
        content:"";
        position:absolute;
        left:0;
        right:0;
        width:100%;
        height:100%;
        --star--opacity:1;
        --star-density:0.8;
        background:paint(yu-show);

    }

  4. 安装http-server

    npm install http-server -g

  5. 在终端命令行中开启http-server

  6. 打开浏览器，输入localhost:8080查看服务。

  7. 在项目根目录下创建sky.js,并在里面写上代码：

    class ShowStar{
    constructor(){

    }
    static get inputProperties(){
        return ["--star--opacity","--star-density"];
    }
    paint(ctx,paintSize,properties){
        let xMax = paintSize.width;
        let yMax = paintSize.height;
        let starOpacity = properties.get("--star--opacity");
        let starDensity = properties.get("--star-density").toString() || 1;
        starDensity > 1 && (starDensity==1);
        let num = Math.round((xMax + yMax) * starDensity);
        //画星星
        for(let i = 0;i < num;i++){
            //随便设置随机数，
            let x = Math.floor(Math.random()*xMax+1);
            let y = Math.floor(Math.random()*yMax+1);
            let size = Math.floor(Math.random()*2+1);
            let o1 = Math.floor(Math.random()*9+1);
            let o2 = Math.floor(Math.random()*9+1);
            let opacity = +("."+(o1+o2))*starOpacity;
            let hue = Math.floor(Math.random()*360+1);
            ctx.fillStyle = `hsla(${hue},30%,80%,${opacity})`
            ctx.fillRect(x,y,size,size);
        }
        ctx.fillRect(0,0,xMax,yMax);
    }
}
registerPaint("show-star",ShowStar);

CSS矩阵：

css3中的旋转，平移，缩放等代码是浏览器暴露给我们的最简单的方法，原始的css代码是:

    transform: matrix(0,0,0,0,0,0,);    
    6个参数从左到右对应的意思是：
    1. 默认值：1，意思是 scaleX 通过设置 X 轴的值来定义缩放。值：>=0
    2. 默认值：0，意思是 skew 倾斜
    3. 默认值：0，意思是 rotate 旋转角度
    4. 默认值：1，意思是 scaleY 通过设置 Y 轴的值来定义缩放。值：>=0
    5. 默认值：0，意思是 translateX 通过设置 X 轴的值来定义左右位移。值：任意
    6. 默认值：0，意思是 translateY 通过设置 Y 轴的值来定义上下位移。值：任意
