Css WorkFlow

webpack管理cssnext和postcss配置书写流程：

  1. 新建项目文件夹。（如cssnexttest）
  2. npm init初始化目录。
  3. npm install webpack --save安装webpack环境
  4. npm install webpack-cli --save安装webpack-cli
  5. 新建src目录，并在目录下创建index.js和index.css
  6. 在项目根目录下创建index.html文件

  7. 在index.css文件中编写cssnext代码

    :root {
    --mainColor:red;/*定义一个父类样式*/
}
.test{
    color:var(--mainColor);/*引用父类样式*/
}
  8. 打开package.json，在scripts字段中添加"dev":"webpack --mode development"
  9. 执行npm run dev命令，生成dist目录。
  10. 在dist目录下创建index.html，引入刚才生成的main.js
  11. 在src目录下的index.js中引入index.css文件。 import item from "./index.css"

  12. 将得到的css值渲染到页面上。

    document.getElementById("app").innerHTML = <h1 class="${item.test}"></h1>//(注意ES6语法的小标点)

  13. 在根目录下创建webpack.config.js
  14. 由于引入了css文件，所以要安装css-loader和style-loader两个包的安装方法：
    1. npm install css-loader --save-dev
    2. npm install style-loader --save-dev

  15. 将如下代码加入到webpack.config.js中

    module.exports = {
  module: {
    rules: [
      {
        test: /\.css$/,
        use: [ 'style-loader', {
              loader: 'css-loader',
              options: {
                modules: true,
                localIdentName: '[path][name]__[local]--[hash:base64:5]'
              }
         } ]
      }
    ]
  }
}

  16. 打开dist目录中的index.html，编写id为app的div元素。

    <body>
    <div id="app"></div>
    <script src="./main.js"></script>
</body>

  17. 执行npm run dev，在浏览器中打开dist目录下的html文件，就可以看到css样式已经引入到了页面中。

  18. 在src下新建components文件夹，在components文件夹下创建main文件夹，将原来src文件夹下的index.css移动到main文件夹下，并重命名成main.css。并在main文件夹下创建main.js文件，代码如下:

    main.css

    :root{
    --mainColor:red;
}
.test{
    color:var(--mainColor);
}

    main.js

    import item from ".main.css";

    原src目录下的index.js

    import item from "./components/main/main.js";

  19. 安装mini-css-extract-plugin

    npm install mini-css-extract-plugin --save-dev

  20. 在webpack.config.js中添加plugins字段，并在最上方引入mini变量,将use字段中的"style-loader替换成miniloader"：

    const MiniCssExtractPlugin = require("mini-css-extract-plugin");        
module.exports = {
    module: {
      rules: [
        {
          test: /\.css$/,
          use: [ {
        loader: MiniCssExtractPlugin.loader
    }, {
                loader: 'css-loader',
               /* options: {
                  modules: true,
                 localIdentName: '[path][name]__[local]--[hash:base64:5]'
                }*/
           } ]
        }
      ]
    },
    plugins: [
        new MiniCssExtractPlugin({
          // Options similar to the same options in webpackOptions.output
          // both options are optional
          filename: "[name].css",
          chunkFilename: "[id].css"
        })
      ]
  }

  21. 运行npm run dev。就会看到在dist目录中生成了main.css,在dist目录中的index.html中引入main.css样式，执行index.html页面，就会看到样式引用到了html页面中。

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
    <link rel="stylesheet" href="./main.css">
</head>
<body>
    <div id="app">
        <h1 class="test">你妹呀</h1>
    </div>
    <script src="./main.js"></script>
</body>
</html>

  22. 安装post-css-loader和postcss-preset-env

    npm install postcss-loader --save-dev
npm install postcss-preset-env --save-dev

  23. 在项目根目录下创建postcss.config.js文件

        文件最终内容：
    const postcssPresetEnv = require('postcss-preset-env');
    module.exports = {
        plugins: {
            "postcss-preset-env":{
                stage: 0,
                features: {
                    'nesting-rules': true
                }
            }
        }
      }

  24. 在webpack.config.js文件中添加postcss-loader字段。

    const MiniCssExtractPlugin = require("mini-css-extract-plugin");
module.exports = {
    module: {
      rules: [
        {
          test: /\.css$/,
          use: [ {
            loader: MiniCssExtractPlugin.loader
          }, {
                loader: 'css-loader',
                options: {
                importLoaders: 1
              //modules: true,
              //localIdentName: '[path][name]__[local]--[hash:base64:5]'
            }
           } ,'postcss-loader']
        }
      ]
    },
    plugins: [
        new MiniCssExtractPlugin({
          // Options similar to the same options in webpackOptions.output
          // both options are optional
          filename: "[name].css",
          chunkFilename: "[id].css"
        })
      ]
  }

  25. 修改main文件夹下的css文件，用cssnext的方式书写。

        :root{
        --mainColor:red;
    }
    .c-main{
        & .test{
            color:var(--mainColor);
        }
    }

  26. 在dist目录下的index.html中引入postcss编辑后的main.css

        <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <meta http-equiv="X-UA-Compatible" content="ie=edge">
        <title>Document</title>
        <link rel="stylesheet" href="./main.css">
    </head>
    <body>
        <div id="app" class="c-main">
            <h1 class="test">卓依婷</h1>
        </div>
        <script src="./main.js"></script>
    </body>
    </html>
  27. 在浏览器中打开index.html，发现被postcss编辑后的cssnext样式已经引入到了页面中。
