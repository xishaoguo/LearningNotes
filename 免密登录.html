免密登录

  1. 生成本机的秘钥对，终端下使用命令：ssh-keygen -t rsa -C "起个任意的名字" -f "任意名字_rsa" （-f是直接输入名字，也可以回车后单独输入）

    ssh-keygen -t rsa -C "起个任意的名字" -f "任意名字_rsa" 
ssh-keygen -t rsa -C "190598130@qq.com" -f "mac_id_rsa"

  2. 将本机刚生成的公钥上传到需要免密登录的服务器。只要看到出现了:Now try logging ...... 就说明已经上传成功了，

    ssh-copy-id -i 生成的公钥 用户名@服务器ip地址或域名 ssh-copy-id -i macidrsa root@www.2xiao.cn

3. 查看上传成功的公钥文件。 1. 登录到远程服务器

    ssh root@www.2xiao.cn
2. 进入到 .ssh目录，执行ls命令就可以看到公钥已被注入到authorized_keys文件中

4. 实现免密登录，在本机的终端下执行命令：ssh -i "私钥文件" "用户名@服务器ip或域名" 就可以实现不用输入密码直接登录上服务器

    ssh -i mac_id_rsa root@www.2xiao.cn

更加快捷的免密登录，（适用于mac和linux）

  1. 修改本地.ssh目录下的config文件，如果没有，就创建。

    Host xiao
User root
HostName www.2xiao.cn
IdentityFile ~/.ssh/mac_id_rsa
Port 22
Protocol 2
Compression yes
ServerAliveInterval 60
ServerAliveCountMax 20
LogLevel INFO

这个文件的格式是固定的:
Host xiao     //起个别名
User root     //远程服务器的用户名
HostName www.2xiao.cn       //远程服务器的ip或者域名
IdentityFile ~/.ssh/mac_id_rsa      //生成的私钥的所在目录
Port 22             //远程服务器的端口号，默认是22,这一行也可以不要
后面的几行照抄

  2. 保存文件后在终端执行:ssh 起的那个别名，就可以登录到远程服务器

    ssh xiao
