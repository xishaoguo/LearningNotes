超级免密登录

当我们实现了免密登录以后，就可以在本机直接执行命令来操作远程服务器了，例如我们想进入远程服务器的根目录，使用命令：

    ssh xiao "cd /"

还可以使用组合命令，执行多条命令，中间用 ; 号隔开

    ssh xiao "cd /;ls"

上面这条命令就可以实现进入到远程服务器的根目录，并且查看目录里面的内容。

如果我们想远程创建一个项目，并且使用npm init初始化这个项目，我们就可以使用命令在本地操作，如：

    ssh xiao "cd yuyu;mkdir jenkinsdemo;cd jenkinsdemo;npm init -y"

知道了上面这些以后，我们就可以开始最后的超级免密登录了。

  1. 新建一个项目，如：sshdemo，初始化这个项目，npm init -y

  2. 安装scripty，这是一个脚本，可以优化我们所写的package.json中的命令。

    npm install --save-dev scripty

  3. 项目目录下创建scripts/dev目录，里面创建hello.sh（经过测试发现必须要有scripts目录，否则不会正常运行）

    //hello.sh
ssh xiao "cd yuyu;mkdir nimei"

  4. 修改package.json

    //package.json
"scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "dev:hello":"scripty"
 },

  5. 安装npm-run-all(这个插件是可以一次性执行多条npm 命令)

    npm install npm-run-all --save-dev

  6. 执行npm run dev:hello就可以启动hello.sh里面的shell脚本命令，从而连接上远程服务器，并创建文件夹,在执行的时候可能会遇到权限问题，我们可以使用sudo chmod -R a+x 文件夹名称 来给文件夹添加权限。如：

    sudo chmod -R a+x scripts

  7. 我们进入远程服务器查看，会发现已经执行了hello.sh里面的命令，已经在服务器上创建了文件夹。

这样做的目的是我们可以在项目里任意的写我们需要在服务器上面的操作了。如：单元测试、e2e测试、babel编译......
